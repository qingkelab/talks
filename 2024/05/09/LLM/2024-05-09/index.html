LLaMA-3еҸҲеҮәжқҘдәҶпјҢз»јеҗҲиЎЁзҺ°йқһеёёжғҠиүіпјҢжҲ‘еңЁе®һйҷ…жөӢиҜ•дёӯиғҪеҠӣд№ҹжҜ”LLaMA-2-7BпјҢMistral-7Bе’ҢGemma-7Bж•ҲжһңеҘҪгҖӮжЁЎеһӢиҝҳжҳҜзӣҙжҺҘеӨҚз”Ёд№ӢеүҚзҡ„д»Јз ҒпјҢдёҚиҝҮжңҖе°Ҹзҡ„8BжЁЎеһӢд№ҹз”ЁдёҠдәҶGQAдәҶпјҢе®һжөӢйҖҹеәҰжҢәеҝ«гҖӮжүӢеӨҙзҡ„llama-2еҸҜд»ҘдёўдәҶпјҢеҸҜд»ҘжӢҘжҠұllama-3дәҶгҖӮ

llama2 еҮәжқҘдәҶпјҢе№¶дё”ејҖжәҗеҸҜе•Ҷз”ЁпјҢиҝҷдёӢејҖжәҗзӨҫеҢәеҸҲиҰҒеҸҳеӨ©дәҶгҖӮеҝ«йҖҹзңӢдёҖдёӢе®ҳзҪ‘д»ҘеҸҠpaperпјҢзңӢзңӢllamav2зӣёжҜ”v1жңүд»Җд№Ҳжӣҙж–°еҗ§пјҡ

  • йў„и®ӯз»ғиҜӯж–ҷд»Һ1->2 Trillion tokens
  • context window й•ҝеәҰд»Һ2048->4096
  • ж”¶йӣҶдәҶ100kдәәзұ»ж ҮжіЁж•°жҚ®иҝӣиЎҢSFT
  • ж”¶йӣҶдәҶ1Mдәәзұ»еҒҸеҘҪж•°жҚ®иҝӣиЎҢRLHF

LLaMAзҺ°еңЁе·Із»ҸжҳҜејҖжәҗзӨҫеҢәйҮҢзӮҷжүӢеҸҜзғӯзҡ„жЁЎеһӢдәҶпјҢдҪҶжҳҜеҺҹж–Үдёӯд»…д»…д»Ӣз»ҚдәҶе…¶е’Ңж ҮеҮҶTransformerзҡ„е·®еҲ«пјҢе№¶жІЎжңүдёҖдёӘе…ЁеұҖзҡ„жЁЎеһӢд»Ӣз»ҚгҖӮжүҫдәҶжүҫе…¶д»–еҚҡе®ўд№ҹйғҪжҳҜе’ҢеҺҹж–ҮдёҖж ·пјҢжІЎжңүд»Ӣз»ҚжЁЎеһӢзҡ„з»“жһ„жҖ»и§ҲгҖӮеӣ жӯӨжү“з®—еҶҷиҝҷзҜҮж–Үз« пјҢдәүеҸ–и®©иҜ»иҖ…дёҚеҸӮиҖғд»»дҪ•е…¶д»–иө„ж–ҷжҠҠLLaMAзҡ„жЁЎеһӢжҗһжҮӮгҖӮ

  • LLaMAеҸҜд»Ҙе°Ҷжӣҙж—©зҡ„KгҖҒVжӢјжҺҘеҲ°еҪ“еүҚKгҖҒVеүҚйқўпјҢеҸҜд»Ҙз”ЁQжҹҘжүҫжӣҙж—©зҡ„дҝЎжҒҜпјҢдёәдәҶжё…жҷ°жІЎеңЁеӣҫдёӯз”»еҮәжқҘгҖӮ

Group Query Attention(V2 only)

иҮӘеӣһеҪ’жЁЎеһӢз”ҹжҲҗеӣһзӯ”ж—¶пјҢйңҖиҰҒеүҚйқўз”ҹжҲҗзҡ„KVзј“еӯҳиө·жқҘпјҢжқҘеҠ йҖҹи®Ўз®—гҖӮеӨҡеӨҙжіЁж„ҸеҠӣжңәеҲ¶(MHA)йңҖиҰҒзҡ„зј“еӯҳйҮҸеҫҲеӨ§пјҢMulti-Query AttentionжҢҮеҮәеӨҡдёӘеӨҙд№Ӣй—ҙеҸҜд»Ҙе…ұдә«KVеҜ№гҖӮGroup Query AttentionжІЎжңүеғҸMQAдёҖж ·жһҒз«ҜпјҢе°ҶqueryеҲҶз»„пјҢз»„еҶ…е…ұдә«KVпјҢж•ҲжһңжҺҘиҝ‘MHAпјҢйҖҹеәҰдёҠдёҺMQAеҸҜжҜ”иҫғгҖӮp.s. иҝҷдёӘжҠҖжңҜfalconе·Із»Ҹз”ЁдёҠдәҶпјҢеҪ“ж—¶falconиҜҙиҮӘе·ұз”Ёзҡ„жҳҜmulti query attentionпјҢеӣ дёәеҪ“group=1ж—¶пјҢGQAе’ҢMQAжҳҜзӯүд»·зҡ„гҖӮfalconж”ҜжҢҒи®ҫзҪ®дёҚеҗҢзҡ„GгҖӮ

RMSNorm

иҝҷжҳҜеңЁBERTгҖҒGPTзӯүжЁЎеһӢдёӯе№ҝжіӣдҪҝз”Ёзҡ„LayerNormпјҡ

\(y=Var(x)+ПөвҖӢxвҲ’Mean(x)вҖӢвҲ—W+B\)

RMSNorm(root mean square)еҸ‘зҺ°LayerNormзҡ„дёӯеҝғеҒҸз§»жІЎд»Җд№Ҳз”Ё(еҮҸеҺ»еқҮеҖјзӯүж“ҚдҪң)гҖӮе°Ҷе…¶еҺ»жҺүд№ӢеҗҺпјҢж•ҲжһңеҮ д№ҺдёҚеҸҳпјҢдҪҶжҳҜйҖҹеәҰжҸҗеҚҮдәҶ40%гҖӮжңҖз»Ҳе…¬ејҸдёәпјҡ

\(y=Mean(x2)+ПөвҖӢxвҖӢвҲ—W\)

жіЁж„ҸйҷӨдәҶжІЎжңүеҮҸеқҮеҖјпјҢеҠ еҒҸзҪ®д»ҘеӨ–пјҢеҲҶжҜҚдёҠжұӮзҡ„RMSиҖҢдёҚжҳҜж–№е·®гҖӮ

LLaMAеңЁ Attention Layerе’ҢMLPзҡ„иҫ“е…ҘдёҠдҪҝз”ЁдәҶRMSNormпјҢзӣёжҜ”еңЁиҫ“еҮәдёҠдҪҝз”ЁпјҢи®ӯз»ғдјҡжӣҙеҠ зЁіе®ҡгҖӮ

SwiGLU

LLaMAжІЎжңүдҪҝз”ЁReLUпјҢиҖҢжҳҜдҪҝз”ЁдәҶSwiGLUпјҢжңүж—¶д№ҹиў«з§°дёәSiLUгҖӮе…¬ејҸдёәпјҡВ \(Sigmoid(x)вҲ—x\)В пјҢж•Ҳжһңзұ»дјје№іж»‘зүҲзҡ„ReLUпјҡ

RoPE

LLaMAдҪҝз”ЁдәҶRotary Position EmbeddingгҖӮеҜ№дәҺQзҡ„з¬¬mдёӘдҪҚзҪ®еҗ‘йҮҸqпјҢйҖҡиҝҮд»ҘдёӢж–№жі•жіЁе…ҘдҪҚзҪ®зј–з Ғпјҡ

\[f(q,m)=вҖӢq0вҖӢq1вҖӢ...qd/2вҲ’1вҖӢqd/2вҖӢqd/2+1вҖӢ...qdвҲ’1вҖӢвҖӢвҖӢГ—вҖӢcos(mОё0вҖӢ)cos(mОё1вҖӢ)...cos(mОёd/2вҲ’1вҖӢ)cos(mОё0вҖӢ)cos(mОё1вҖӢ)...cos(mОёd/2вҲ’1вҖӢ)вҖӢвҖӢ+вҖӢвҲ’qd/2вҖӢвҲ’qd/2+1вҖӢ...вҲ’qdвҲ’1вҖӢq0вҖӢqd1вҖӢвҖӢ...qd/2вҲ’1вҖӢвҖӢвҖӢГ—вҖӢsin(mОё0вҖӢ)sin(mОё1вҖӢ)...sin(mОёd/2вҲ’1вҖӢ)sin(mОё0вҖӢ)sin(mОё1вҖӢ)...sin(mОёd/2вҲ’1вҖӢ)вҖӢвҖӢ\]

е…¶дёӯВ ОёВ жҳҜеҖјд»ӢдәҺ[1,0)д№Ӣй—ҙзҡ„еӣәе®ҡеҗ‘йҮҸгҖӮйҖҡиҝҮд»ҘдёӢд»Јз Ғеҫ—еҲ°дәҶдёҠејҸдёӯзҡ„з¬¬дәҢйЎ№ \(cos(mОёiвҖӢ)\)В  е’Ңз¬¬еӣӣйЎ№ В \(sin(mОёiвҖӢ)\) В гҖӮ

еҗҺйҖҡиҝҮд»ҘдёӢд»Јз Ғеҫ—еҲ°з»“еҗҲдәҶдҪҚзҪ®зј–з Ғзҡ„Q,K(Kе’ҢQдҪҝз”ЁеҗҢж ·зҡ„ж–№ејҸиҝӣиЎҢдҪҚзҪ®зј–з Ғ)гҖӮ

дҪҝз”ЁдәҶиҝҷд№ҲеӨҚжқӮзҡ„дҪҚзҪ®зј–з ҒпјҢжңүд»Җд№ҲеҘҪеӨ„е‘ўпјҹд»ҺдёҠйқўзҡ„е…¬ејҸеҸҜд»ҘзңӢеҮәпјҢRoPEеҪўејҸдёҠжҳҜз»қеҜ№дҪҚзҪ®зј–з ҒпјҢеҚідҫқиө–е…¶з»қеҜ№дҪҚзҪ®mгҖӮ

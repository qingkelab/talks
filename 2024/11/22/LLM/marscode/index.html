Why MarsCode IDE?

MarsCode IDEжҳҜMarscodeзҡ„йҮҚзЈ…дә§е“ҒпјҢд№ҹжҳҜжҲ‘жғійҮҚзӮ№е’ҢеӨ§е®¶еҲҶдә«зҡ„, дёҖдёӘеҶ…зҪ®AIзј–з ҒеҠ©жүӢзҡ„дә‘з«ҜIDEе№іеҸ°гҖӮжҳҜеӣҪеҶ…йҰ–ж¬ҫдә‘з«ҜAI IDEдә§е“ҒпјҢе®ғе°ҶAIе’Ңдә‘и®Ўз®—е®ҢзҫҺз»“еҗҲпјҢдёәејҖеҸ‘иҖ…еёҰжқҘдәҶеүҚжүҖжңӘжңүзҡ„дҫҝеҲ©,е…·жңүиҜёеӨҡдјҳеҠҝгҖӮ

  • дёҚз”Ёй…ҚзҪ®зҺҜеўғ: еҜ№дәҺе…Ҙй—ЁйҖүжүӢжқҘиҜҙйқһеёёдёҚй”ҷпјҢжңүиҝҮејҖеҸ‘з»ҸйӘҢзҡ„еҗҢеӯҰйғҪзҹҘйҒ“пјҢжҗӯе»әзҺҜеўғжҳҜеҫҲиҖ—ж—¶зҡ„пјҢеҰӮжһңжҠҘй”ҷдәҶпјҢеҜ№дәҺж–°дәәиҝҳжҳҜеҫҲйҡҫи§Јзҡ„пјҢ зҺ°еңЁдёҚз”Ёжҗӯе»әзҺҜеўғпјҢеҸҜд»ҘзӣҙжҺҘиҝӣе…ҘејҖеҸ‘дәҶпјҢ ж„ҹи§үеҜ№зј–зЁӢж•ҷиӮІдёҠд№ҹжҳҜдёҖеӨ§еҘҪеӨ„пјҢе°Өе…¶еҜ№дәҺе°‘е„ҝгҖҒж–°жүӢзј–зЁӢ...
  • ж–№дҫҝжј”зӨәDemo: жңүж—¶еҖҷжңӢеҸӢи®©её®еҝҷеҒҡдёӘеҠҹиғҪпјҢз„¶еҗҺиҮӘе·ұеҶҷе®ҢдәҶд»Јз ҒпјҢиҝҳиҰҒйғЁзҪІдёҖдёӘжңҚеҠЎпјҢдҪҶжҳҜеҰӮжһңдёҚеңЁеұҖеҹҹзҪ‘пјҢжңӢеҸӢж №жң¬е°ұи®ҝй—®дёҚдәҶдҪ зҡ„еҠҹиғҪпјҢиҝҳиҰҒеҺ»еј„жңҚеҠЎеҷЁдёҖеӨ§е ҶдёңиҘҝд№ҹйқһеёёзҡ„йә»зғҰпјҢзҺ°еңЁз”ЁMarsCode IDEпјҢеҸҜд»ҘзӣҙжҺҘеҗҜеҠЁжңҚеҠЎпјҢи®©иҝңзЁӢзҡ„жңӢеҸӢд№ҹиғҪзӣҙжҺҘдҪҝз”ЁдҪ зҡ„еҠҹиғҪпјҢйқһеёёж–№дҫҝпјӣ
  • е°ҸеҠҹиғҪи„ҡжң¬з®ЎзҗҶ: ејҖеҸ‘еҗҢеӯҰпјҢжңүж—¶еҖҷдјҡеҶҷзӮ№и„ҡжң¬дёҙж—¶и§ЈеҶідёӢжҹҗдәӣеҠҹиғҪпјҢжң¬ең°зҡ„и„ҡжң¬жңүж—¶еҖҷеӣӣж•Јеҗ„еӨ„пјҢеҗҺйқўйғҪеҝҳдәҶеңЁе“ӘйҮҢпјҢжҲ‘и§үеҫ—IDEиҝҳеҸҜд»Ҙе……еҪ“жҲ‘д»¬дёҖдәӣе°ҸеҠҹиғҪи„ҡжң¬зҡ„еӯҳеӮЁең°ж–№пјҢжҠҠеҗ„з§Қеҗ„ж ·зҡ„е°ҸеҠҹиғҪжүҳз®ЎеҲ°дәҶIDEдёҠгҖӮз»ҹдёҖз®ЎзҗҶпјҢз»“еҗҲAIзј–зЁӢеҠ©жүӢеё®дҪ еҶҷи„ҡжң¬пјҢжғіжғід№ҹдёҚй”ҷпјӣ
  • жҸ’д»¶еҠҹиғҪпјҡеӨ§е®¶еҸҜиғҪжӣҙзҶҹжӮүдәҶпјҢMarsCodeиҝҳж”ҜжҢҒдёҖдәӣжҸ’д»¶пјҢжҸ’д»¶е°ұжҳҜеҸҜд»ҘеҒҡеҗ„з§Қеҗ„ж ·зҡ„дәӢжғ…пјҢеңЁдә‘з«ҜеҒҡеҗ„з§Қеҗ„ж ·зҡ„дәӢжғ…пјҢеӨ§е®¶еҸҜд»Ҙе°қиҜ•е®үиЈ…дёӢйҮҠж”ҫдёӢжғіиұЎеҠӣгҖӮ

еҰӮжһңиҜҙжҳҜдёҖдәӣеӨ§еһӢйЎ№зӣ®зҡ„еҚҸдҪңдёҠпјҢзҺ°еңЁ IDEеҸҜд»Ҙе°қиҜ•дёӢпјҢдҪҶжҳҜи§үеҫ—еҸҜиғҪжң¬ең°жӣҙж–№дҫҝпјҢеҜ№дәҺдёҖдәӣе°ҸеһӢзҡ„зј–зЁӢеҚҸдҪңеҠӣеңәжҷҜпјҢж„ҹи§үжҢәжңүж•Ҳзҡ„гҖӮ

дёҚиҝҮпјҢMarsCode зӣ®еүҚд№ҹеӯҳеңЁдёҖдәӣйҷҗеҲ¶

  • йЎ№зӣ®е®№йҮҸжңүйҷҗ: еҸӘиғҪеҲӣе»ә10дёӘйЎ№зӣ®пјҢдҪҶжҳҜеҹәжң¬дёҠи¶іеӨҹдәҶпјӣ

  • иө„жәҗе°Ҹ: зӣ®еүҚжІЎжңүжң¬ең°йӮЈд№ҲеӨ§пјҢеҚ•дёӘйЎ№зӣ®еҸҜд»ҘдҪҝз”Ёзҡ„жңҖеӨ§иө„жәҗеҰӮдёӢпјҡ

  • еҸҜи§ҶеҢ–й—®йўҳ: дёҖдәӣйңҖиҰҒеұ•зӨәзҡ„жҜ”еҰӮз»ҳеҲ¶еӣҫзҡ„ж— жі•еҒҡеҲ°гҖӮ

иҝӣе…ҘMarscode IDE

иҝӣе…ҘеҗҺ, йҰ–ж¬Ўиҝӣе…Ҙзҡ„з”ЁжҲ·дјҡиў«иҰҒжұӮи®ҫзҪ®дёӘдәәдҝЎжҒҜ, еҰӮдёӢжүҖзӨә, дҝқеӯҳжҲҗеҠҹеҗҺ, жҲ‘д»¬е°ұеҸҜд»ҘдҪҝз”ЁдәҶ!

еңЁеҲӣе»әйЎ№зӣ®ж—¶пјҢе®ҡд№үеҘҪйЎ№зӣ®еҗҚз§°пјҲиҝҷжҳҜеҝ…еЎ«йЎ№е“ҰпјүпјҢиҝҳеҸҜд»Ҙж №жҚ®йЎ№зӣ®зү№зӮ№иЎҘе……иҜҰз»Ҷзҡ„йЎ№зӣ®жҸҸиҝ°пјҲйҖүеЎ«йЎ№пјүпјҢйҡҸеҗҺзӮ№еҮ»еҲӣе»әпјҢдёҖдёӘдё“еұһзҡ„зј–зЁӢйЎ№зӣ®з©әй—ҙе°ұиҜһз”ҹдәҶгҖӮ

жӯӨеӨ–пјҢйҷӨдәҶд»Һйӣ¶ејҖе§ӢеҲӣе»әйЎ№зӣ®пјҢдҪ иҝҳеҸҜд»ҘеҜје…Ҙ Git д»Јз Ғд»“еә“пјҢж–№дҫҝж•ҙеҗҲе’Ңз®ЎзҗҶе·Іжңүд»Јз Ғиө„жәҗпјҢиҝҷдёҖеҠҹиғҪжҲ‘д»¬еҗҺз»ӯдјҡиҜҰз»Ҷд»Ӣз»ҚгҖӮ

算法研究类岗位

多模态算法工程师

公司：AIGC创业公司 - 2亿参数端侧大模型 - 对话助手 - 清华NLP LAB

工作地点：北京海淀
薪资：面议

职位描述： 1. 负责多模态大模型预训练、SFT、对齐等各阶段的模型开发与训练，涉及模型高效结构、多模态 scaling law、 数据配比、性能优化等探索; 2. 参与多模态大模型的训练加速和高效部署，包括但不限于于系统调优、硬件适配等工作 3. 参与前沿多模态算法与应用的调研和研究创新工作，并将多模态 AI 能力应用在产品中。

任职要求： 1. 硕士及以上学历，三年以上算法开发经验，一年以上大模型相关经验； 2. 对多模态大模型有深入理解和实践经验优先; 有多机多卡、大规模集群训练经验者优先；有多模态视频理解经验者优先； 3. 熟悉多模态大模基本原理，至少熟悉一种深度学习编程框架，如PyTorch等，有模型开发、部署、微调经验; 熟悉掌握分布式训练框架如 DeepSpeed/Megatron； 4. 良好的编程风格与文档撰写能力，有多人协作开发经验，对代码有严格的质量要求；熟悉常见开发规范，熟悉 Git 等开发工具，熟悉代码库的创建与维护操作; 5. 及时跟进前沿技术进展，有优秀的问题分析和解决能力以及团队协作能力。

视频算法

公司：AIGC创业公司 - 生图，生视频 - B端电商 - C端创作者

工作地点：北京海淀/上海徐汇
薪资：面议

职位描述： 1. 负责技术方案的落地实施：构建端到端的算法流程，将技术方案转化为可落地的产品，以满足市场需求和业务目标； 2. 负责核心模型的完善和维护：进行核心模型的算法优化，提升图像/视频生成模型在生成质量、多样性、可控性和采样效率等方面的性能； 3. 优化提升各模型业务能力：优化多模态大模型和视觉感知类模型，提高模型精度、性能和泛化能力，满足复杂的业务需求； 4. 开发创新产品：生成和感知类模型的创新改造，结合前沿学术成果，打造具有市场竞争力的差异化产品。

任职要求： 1. Base北京/上海，工作经验 1-5 年；接受优秀应届毕业生，有3个月以上大厂或知名创业公司实习经历； 2. 硕士及以上学历，计算机、数学等相关专业，熟练掌握计算机视觉和机器学习算法基础； 3. 具备扎实的编程功底，熟悉Linux开发环境，熟练使用C++/Python等常用语言中的一种； 4. 熟练掌握常见CV感知类模型的方法和技术细节，有实际业务落地经验，作为核心贡献者长期打磨关键模型； 5. 熟悉经典生成式模型的原理和技术细节，包括但不限于SD系列、DALLE系列等； 6. 熟悉经典多模态模型的原理和技术细节，包括但不限于CLIP、BLIP、LLAVA等； 7. 了解掌握学术界、工业界前沿技术，熟悉最新方法细节，具备利用各类技术创新建构的差异化能力，并解决实际业务场景的问题； 8. 具备良好的沟通能力、团队精神和业务思维； 9. 拥有ACM区域赛奖牌，在CV和ML顶级会议期刊上发表相关论文者优先；

AIGC资深算法，视频生成

公司：AIGC创业公司 - 字节大佬创业

工作地点：北京/上海
薪资：面议

职位描述： 1. 负责视觉生成算法研发，包括但不限于图片生成、视频生成、3D内容生成等； 2. 在自监督学习、表征学习、扩散模型等领域开展前沿技术探索和研究，包括但不限于对比学习、掩码学习、自回归学习、跨模态表征对齐、扩散生成等方向； 3. 与工程、产品团队深度合作，落地成熟的AIGC算法，提供极致的用户体验。

任职要求： 1. 本科及以上学历，计算机相关专业； 2. 扎实的编程基础，熟练掌握C++/Python； 3. 了解主流 AIGC 算法模型，有 AIGC 模型推理和训练加速落地经验者优先； 4. 有大模型训练优化经验，熟悉FSDP/DeepSpeed/Megatron等分布式训练框架经验者优先； 5.北京/上海均可。

推理加速算法工程师

公司：幻方 - 万卡集群

工作地点：北京/杭州
薪资：面议

职位描述： 1. 设计开拓性的新的深度神经网络 2. 构建科学、严谨的算法评测体系 3. 紧跟领域前沿，推动基础研究

任职要求： 1. 精通机器学习（深度学习），具备创新研究能力。 2. 编程能力出色，熟练掌握至少两种编程语言，熟练掌握Tensorflow/Pytorch。 3. 有丰富的研究成果，在国际顶会或期刊发表相关论文（包括但不限于NIPS, ICML, CVPR, COLT)。 4. 在领域内知名比赛取得优异成绩者优先。 5. 认同开放共进的企业文化，积极创新，乐于挑战，良好的逻辑思维、沟通协调和自我学习能力，主动负责，严谨细致，勤奋踏实。

工程类岗位

GPU性能优化工程师

公司：自动驾驶
工作地点：广州/深圳/上海/新加坡
薪资：面议

职位描述： 作为一名GPU AI/HPC专家工程师，你将在设计和实施突破性的GPU计算方面提供专业支持，去解决深度学习、高性能计算和计算密集型工作负载相关的问题，优化GPU计算中的容量管理和分配。

主要职责： 1. 性能分析与优化：深入分析和优化GPU应用程序，识别并消除性能瓶颈，包括内存访问模式、线程调度和执行效率。 2. 算法开发：利用CUDA或DirectX等并行计算框架，开发和实现高效的GPU加速算法，提升计算密集型任务的处理速度。 3. 向团队提供技术指导和培训，分享GPU编程的最佳实践和优化技巧。 4. 持续关注GPU技术和行业趋势，评估并整合新技术以提升系统性能。

任职要求： 1. 计算机科学、电子工程或相关领域的学士或硕士学位及以上。 2. 至少5年的GPU编程和性能优化经验，具有丰富的CUDA或DirectX开发背景。 3. 精通C/C++编程，具备深入的计算机体系结构和并行计算原理知识。 4. 熟练使用GPU性能分析和调试工具，具有实际的性能优化案例和成果。 5. 具备分析和调整各种AI/HPC工作负载性能的经验。

加分项： 1. 具备高性能计算（HPC）领域的相关研究经验。 2. 对机器学习框架如TensorFlow或PyTorch有深入的GPU优化经验。 3. 使用Orin或Xavier平台进行AI模型部署和优化，以及开发自动驾驶相关算法的实战经验。 4. 具有NVIDIA GPU和CUDA编程的经验。

CPU算法优化工程师

公司：自动驾驶
工作地点：广州/深圳/上海/新加坡
薪资：面议

职位描述： 作为CPU算法优化工程师，您将致力于提升软件应用在中央处理器上的执行效率和性能。您将通过优化算法和改进代码实现，确保我们的应用程序能够充分利用CPU的计算能力。您的角色将在我们的任务中发挥关键作用，最大化我们CPU的潜力。

主要职责： 1. 分析现有软件应用的CPU使用情况，识别性能瓶颈。 2. 设计和实施算法优化策略，包括但不限于algorithm implementation, 循环优化、分支预测改进、指令级并行性增强等。 3. 利用多线程和并行处理技术提高应用程序的计算效率。 4. 与软件开发团队合作，将性能优化集成到软件开发生命周期中。 5. 编写技术文档，包括优化报告、性能分析和最佳实践。 6. 跟踪最新的CPU架构发展和性能优化技术，分析和优化算法模型理论及实际性能。

任职要求： 1. 计算机科学、人工智能、应用数学或相关领域的硕士或博士学位及以上 2. 至少5年的软件开发经验，具有CPU性能优化的实际案例。 3. 精通C++编程语言。 4. 熟悉计算机和CPU工作原理

加分项： 1. 具有编译器优化经验。 2. 对并行计算、多线程编程有深入理解。 3. 在ACM竞赛获奖（ICPC） 4. 硕士/博士研究方向为Aigorithms 5. 熟悉OpenCV代码实现细节

推理加速算法工程师

公司：幻方 - 万卡集群

工作地点：北京/杭州
薪资：面议

主要职责： 1. 优化大规模AIGC模型服务全链路性能，打造低延时、高吞吐、高稳定性的推理系统； 2. SD/LLM模型推理优化加速，包括但不限于算子优化，模型量化，模型裁剪，模型蒸馏等； 3. AIGC大模型分布式训练和加速优化。

任职要求： 1. 了解机器学习（深度学习）的各类模型，具备较强的工程能力。 2. 编程能力出色，熟练掌握Python和C++，掌握Pytorch/Tensorflow。 3. 熟练掌握深度学习各类模型架构的使用和设计。 4. 在领域内知名比赛取得优异成绩者优先。
